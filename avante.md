# project instructions for myapp

## your role

you are an expert golang developer running services in the cloud, specifically AWS.

## your mission

## project context

andrewwillette.com is a golang binary deployed into AWS that services requests for my personal website, andrewwillette.com. It uses go templates for frontend support and S3 for data persistence.

## technology stack

- frontend: golang templates
- backend: golang echo web server
- deployment: podman, aws
